## Getting Started

### Accessing Enhanced PMR

1. Navigate to **Reports** > **Enhanced PMR** from the main menu
2. You'll see your existing reports or the option to create a new one
3. Click **Create New PMR** to start a new monthly report

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Active internet connection for real-time features
- Appropriate project permissions (Editor or Admin role)

## Creating a New PMR

### Step 1: Select Template

1. Choose from AI-suggested templates based on your project type
2. Preview template sections and structure
3. Customize template if needed
4. Click **Use Template** to proceed

### Step 2: Configure Report Settings

- **Report Month**: Select the reporting period
- **Project**: Choose the project for this report
- **Distribution List**: Add stakeholders who will receive the report
- **Access Level**: Set viewing and editing permissions

### Step 3: Generate Initial Content

The system will automatically:
- Pull real-time project metrics
- Generate AI-powered executive summary
- Create section templates with suggested content
- Identify key insights and alerts

## Using the PMR Editor

### Editor Interface

The PMR editor consists of three main panels:

1. **Editor Panel** (Center): Rich text editor for report content
2. **AI Insights Panel** (Right): AI-generated insights and recommendations
3. **Collaboration Panel** (Right): Active users, comments, and conflicts

### Editing Sections

#### Section Structure

Each report is organized into sections:
- Executive Summary
- Budget Status
- Schedule Performance
- Resource Allocation
- Risk Management
- Key Achievements
- Issues and Concerns
- Next Period Outlook

#### Editing Content

1. Click on any section to expand it
2. Use the rich text toolbar for formatting:
   - **Bold**, *Italic*, `Code`
   - Headings (H1, H2, H3)
   - Bullet and numbered lists
   - Blockquotes
3. Content auto-saves as you type
4. Click **Save Section** to manually save

#### AI Assistance

While editing, you can:
- Click **Get AI Suggestions** for content recommendations
- View confidence scores for AI-generated content
- Accept or modify AI suggestions
- Request specific insights for any section

### Formatting Tools

#### Text Formatting

## AI-Powered Insights

### Understanding Insights

AI insights are categorized into four types:

1. **Alerts**: Critical issues requiring immediate attention
2. **Predictions**: Forecasts based on current trends
3. **Recommendations**: Suggested actions for improvement
4. **Observations**: Notable patterns or trends

### Insight Categories

- **Budget**: Financial performance and variance analysis
- **Schedule**: Timeline predictions and critical path analysis
- **Resources**: Capacity planning and allocation optimization
- **Risk**: Risk identification and mitigation strategies

### Working with Insights

#### Viewing Insights

1. Insights appear automatically in the AI Insights Panel
2. Each insight shows:
   - Title and description
   - Confidence score (0-100%)
   - Supporting data
   - Predicted impact
   - Recommended actions
   - Priority level (High, Medium, Low)

#### Validating Insights

1. Review the insight details
2. Click **Validate** if the insight is accurate
3. Click **Not Accurate** if the insight needs correction
4. Add notes to explain your validation decision

#### Applying Insights

1. Click **Apply** on any insight
2. The system will:
   - Update relevant report sections
   - Add supporting data and charts
   - Include recommended actions
3. Review and modify the applied content as needed

#### Generating New Insights

1. Click **Generate Insights** in the AI Insights Panel
2. Select specific categories (optional)
3. Wait for AI analysis to complete
4. Review and validate new insights

### Insight Confidence Scores

- **90-100%**: High confidence - Backed by strong data patterns
- **70-89%**: Medium confidence - Supported by available data
- **50-69%**: Low confidence - Limited data or uncertain patterns
- **Below 50%**: Very low confidence - Requires manual verification

## Real-Time Collaboration

### Collaborative Editing

Multiple users can edit the same report simultaneously:

#### Active User Indicators

- See who's currently viewing/editing the report
- View user cursors and selections in real-time
- Identify which sections are being edited

#### Cursor Tracking

- Each user has a unique color-coded cursor
- Hover over a cursor to see the user's name
- Cursors show real-time editing positions

#### Conflict Resolution

When conflicts occur:

1. **Conflict Detection**: System automatically detects conflicting edits
2. **Conflict Alert**: You'll see a warning banner on affected sections
3. **Resolution Options**:
   - Keep your version
   - Accept other user's version
   - Merge both versions manually
4. **Review Changes**: Compare versions side-by-side
5. **Resolve**: Click **Resolve Conflict** to finalize

### Comments and Discussions

#### Adding Comments

1. Select text in any section
2. Click the **Comment** button
3. Type your comment
4. Tag users with @mention
5. Click **Post Comment**

#### Replying to Comments

1. Click on any comment thread
2. Type your reply
3. Click **Reply**

#### Resolving Comments

1. Once a comment is addressed
2. Click **Resolve** to mark it complete
3. Resolved comments are archived but can be viewed

### Presence Indicators

The Collaboration Panel shows:
- Active users (green dot)
- Away users (yellow dot)
- Offline users (gray dot)
- Last activity timestamp

## Exporting Reports

### Export Formats

Enhanced PMR supports multiple export formats:

1. **PDF**: Professional print-ready documents
2. **Excel**: Data-rich spreadsheets with charts
3. **PowerPoint**: Presentation-ready slides
4. **Word**: Editable documents

### Export Process

#### Step 1: Configure Export

1. Click **Export** in the toolbar
2. Select export format
3. Choose template style
4. Configure options:
   - Include/exclude sections
   - Add charts and visualizations
   - Include AI insights
   - Add appendices

#### Step 2: Customize Output

- **PDF Options**:
  - Page size (Letter, A4, Legal)
  - Orientation (Portrait, Landscape)
  - Include table of contents
  - Add page numbers
  - Include watermark

- **Excel Options**:
  - Include raw data sheets
  - Add pivot tables
  - Include charts
  - Format cells

- **PowerPoint Options**:
  - Slide layout
  - Include speaker notes
  - Add animations
  - Include data tables

- **Word Options**:
  - Document style
  - Include comments
  - Add track changes
  - Include revision history

#### Step 3: Generate and Download

1. Click **Generate Export**
2. Wait for processing (usually 10-30 seconds)
3. Download file when ready
4. File is also saved to export history

### Export Queue

- View export status in real-time
- Download previous exports
- Cancel pending exports
- Re-generate failed exports

### Scheduled Exports

1. Click **Schedule Export**
2. Set frequency (Weekly, Monthly, Quarterly)
3. Choose recipients
4. Configure delivery method (Email, Shared Drive)
5. Set start date and time

## Troubleshooting

### Common Issues

#### Report Not Loading

**Problem**: Report page shows loading spinner indefinitely

**Solutions**:
1. Refresh the page (Ctrl/Cmd + R)
2. Clear browser cache
3. Check internet connection
4. Verify project permissions
5. Contact support if issue persists

#### Real-Time Collaboration Not Working

**Problem**: Can't see other users or changes aren't syncing

**Solutions**:
1. Check connection status indicator (top right)
2. Refresh the page to reconnect
3. Verify WebSocket connection isn't blocked by firewall
4. Check browser console for errors
5. Try a different browser

#### AI Insights Not Generating

**Problem**: "Generate Insights" button doesn't work

**Solutions**:
1. Ensure project has sufficient data
2. Check that AI services are available (status indicator)
3. Try generating insights for specific categories
4. Wait a few minutes and retry
5. Contact support if issue persists

#### Export Fails

**Problem**: Export generation fails or times out

**Solutions**:
1. Reduce report size by excluding sections
2. Try a different export format
3. Check export queue for errors
4. Ensure report is saved before exporting
5. Contact support for large reports

#### Editing Conflicts

**Problem**: Frequent editing conflicts with other users

**Solutions**:
1. Coordinate editing times with team
2. Assign sections to specific users
3. Use comments instead of direct edits
4. Work in preview mode when not editing
5. Resolve conflicts promptly

### Performance Issues

#### Slow Editor

**Problem**: Editor is laggy or unresponsive

**Solutions**:
1. Close unused browser tabs
2. Disable browser extensions
3. Use a modern browser (latest version)
4. Reduce number of open sections
5. Clear browser cache

#### Slow Insight Generation

**Problem**: AI insights take too long to generate

**Solutions**:
1. Generate insights for specific categories only
2. Ensure project data is up to date
3. Check system status for AI service availability
4. Try during off-peak hours
5. Contact support if consistently slow

## Glossary

**AI Insight**: Machine learning-generated analysis and recommendations based on project data

**Confidence Score**: Numerical indicator (0-100%) of AI prediction accuracy

**Collaboration Session**: Real-time editing session with multiple users

**Conflict**: Simultaneous edits to the same content by different users

**Cursor Position**: Real-time indicator of where other users are editing

**Export Pipeline**: System for generating reports in multiple formats

**Monte Carlo Analysis**: Statistical simulation for predictive analytics

**PMR Section**: Individual component of a monthly report (e.g., Executive Summary)

**Real-Time Metrics**: Live project data updated continuously

**Template**: Pre-configured report structure and content

**Validation**: User confirmation of AI insight accuracy

**WebSocket**: Technology enabling real-time bidirectional communication
