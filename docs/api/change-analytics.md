# Change Analytics API

See [change-orders.md](./change-orders.md) for full API documentation. Key analytics endpoints:

- `GET /change-analytics/metrics/{project_id}` - Metrics
- `GET /change-analytics/trends/{project_id}` - Trends
- `GET /change-analytics/dashboard/{project_id}` - Dashboard data
- `POST /change-analytics/reports/{project_id}` - Generate report
