# Enhanced PMR Quick Reference Guide

## Quick Start

### Creating Your First Report
1. Navigate to **Reports** > **Enhanced PMR**
2. Click **Create New PMR**
3. Select a template
4. Configure report settings
5. Click **Generate** to create initial content

### Essential Actions
- **Save**: Ctrl/Cmd + S or click Save button
- **Generate AI Insights**: Click "Generate Insights" in AI panel
- **Export**: Click Export button, select format, configure, download
- **Collaborate**: Share report link with team members

## Templates

### Standard Templates
- **Executive PMR**: High-level summary for executives
- **Detailed PMR**: Comprehensive report with all metrics
- **Construction PMR**: Tailored for construction projects
- **IT Project PMR**: Focused on software/IT deliverables
- **Financial PMR**: Emphasis on budget and financial metrics

### Using Templates
1. Click **Templates** button
2. Browse or search templates
3. Preview template structure
4. Click **Use Template**
5. Customize as needed

### Creating Custom Templates
1. Configure your report structure
2. Add custom sections and fields
3. Set default formatting
4. Click **Save as Template**
5. Name and describe your template

## Common Tasks

### Generate AI Content for Empty Section
1. Click on empty section
2. Click **Get AI Suggestions**
3. Review generated content
4. Accept, modify, or regenerate
5. Save section

### Add a New Section
1. Click **+ Add Section**
2. Enter section title
3. Choose section type (optional)
4. Add content or generate with AI
5. Save section

### Reorder Sections
1. Hover over section header
2. Click and hold drag handle
3. Drag to new position
4. Release to drop
5. Changes save automatically

### Share Report with Team
1. Click **Share** button
2. Enter team member emails
3. Set permissions (View/Edit)
4. Add message (optional)
5. Click **Send Invitations**

### Schedule Recurring Export
1. Click **Export** > **Schedule**
2. Select format and template
3. Choose frequency (Weekly/Monthly)
4. Add recipients
5. Set start date and time
6. Click **Schedule**

## Troubleshooting

### Report Not Loading
- Refresh page (Ctrl/Cmd + R)
- Clear browser cache
- Check internet connection
- Verify project permissions

### Real-Time Sync Issues
- Check connection status (top right)
- Refresh page to reconnect
- Verify WebSocket not blocked
- Try different browser

### AI Insights Not Generating
- Ensure project has sufficient data
- Check AI service status
- Try specific categories only
- Wait and retry after a few minutes

### Export Fails
- Reduce report size
- Try different format
- Check export queue
- Ensure report is saved first
