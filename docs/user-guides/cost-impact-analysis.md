# Cost Impact Analysis User Guide

Cost impact analysis supports change order pricing with direct costs, indirect costs, and scenario comparison.

## Direct Costs

Labor, material, equipment, subcontract, and other costs entered as line items with quantity and unit rates.

## Indirect Costs

Overhead, profit, and contingency applied as percentages to direct costs.

## Scenarios

- **Optimistic** – Lower bound estimate
- **Most Likely** – Expected value
- **Pessimistic** – Upper bound estimate

Use scenario comparison for risk assessment and contingency planning.
