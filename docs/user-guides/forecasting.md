# Forecasting User Guide

## Overview

Project Controls forecasting provides month-by-month cost projections and scenario analysis.

## Monthly Forecast

The monthly forecast distributes remaining budget across future periods based on planned schedule and resource utilization. It can include risk adjustments and contingency.

## Scenarios

- **Best case**: Optimistic assumptions, lower costs
- **Most likely**: Expected outcomes
- **Worst case**: Pessimistic assumptions, higher costs

Use scenario comparison for risk assessment and contingency planning. Export forecasts for reporting and stakeholder communication.
